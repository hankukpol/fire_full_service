## 1. 소방공무원 채용 전체 흐름

```
[공고] → [접수] → [필기시험] → [체력시험] → [면접] → [최종합격]
                       ↑
               이 서비스가 다루는 구간
               (필기 합격예측)
```

### 최종 합격자 성적 반영 비율

| 채용 유형 | 필기 | 체력 | 면접 |
|---------|------|------|------|
| 공채 | **50%** | 25% | 25% |
| 경채 (체력시험 있음 — 구조·구급 등) | **50%** | 25% | 25% |
| 경채 (체력시험 없음 — 일부 분야) | **75%** | — | 25% |

> **서비스 범위**: 현재는 **필기시험 단계**만 다룹니다.
> 체력·면접 환산 기능은 미구현 (추후 확장 가능).

## 2. 채용유형 4종 완전 정리

소방은 크게 **공채**와 **경채**로 나뉘며, 경채는 다시 3종으로 세분화됩니다.

### 2026년 전체 채용인원

| ExamType | 구분 | 전국 합계 | 성별 선발 |
|----------|------|---------|---------|
| `PUBLIC` | **공채** | 1,444명 (남 1,313 + 여 131) | 남/여 **분리** 선발 |
| `CAREER_RESCUE` | **구조 경채** | 226명 | **남자만** |
| `CAREER_ACADEMIC` | **소방관련학과 경채** | 101명 (남 51 + 여 37 + 양성 13) | 지역마다 다름 |
| `CAREER_EMT` | **구급 경채** | 1,043명 (남 734 + 여 309) | 남/여 **분리** 선발 |
| **합계** | — | **2,814명** | — |

## 5. 필기 합격배수 ★ (핵심)

필기시험은 최종 선발인원의 **N배수**만큼만 합격시킵니다.
(나머지는 체력·면접을 보기 위한 후보)

### 5-1. 공채 합격배수

| 선발예정인원 | 필기 합격자 수 |
|----------|------------|
| 1~10명 | 선발인원 × **3배** |
| 11~20명 | 선발인원 × **2.5배** |
| 21~50명 | 선발인원 × **2배** |
| **51명 이상** | 선발인원 × **1.5배** |

**예시**: 공채 대구 남자 선발인원 48명
→ 21~50명 구간 → 합격배수 **2배**
→ 필기 합격자 = ceil(48 × 2.0) = **96명**

### 5-2. 경채 합격배수 (구조/소방학과/구급 공통)

| 선발예정인원 | 필기 합격자 수 |
|----------|------------|
| 1명 | **3명** |
| 2명 | **6명** |
| 3명 | **8명** |
| 4명 | **9명** |
| 5명 | **10명** |
| 6~50명 | 선발인원 × **1.8배** |
| **51명 이상** | 선발인원 × **1.5배** |

**예시**: 구급 경채 대구 남자 선발인원 35명
→ 6~50명 구간 → 합격배수 **1.8배**
→ 필기 합격자 = ceil(35 × 1.8) = **63명**

### 5-3. 동점자 처리

합격선 동점자는 선발 배수 초과하더라도 **전원 합격**.

> **코드**: `src/lib/prediction.ts` → `getPublicPassMultiple()`, `getCareerPassMultiple()`

## 6. 합격예측 피라미드 등급

서비스에서 5단계로 합격 가능성을 시각화합니다.

| 등급 | 기준 | 의미 |
|------|------|------|
| **확실권** | 내 순위 ≤ 모집인원 | 1배수 이내, 거의 확실 |
| **유력권** | 내 순위 ≤ 모집인원 × 1.2 | 안전권 내 |
| **가능권** | 내 순위 ≤ 합격배수 기준 인원 | 필기 합격 커트 이내 |
| **도전권** | 내 순위 > 합격배수 기준 인원 | 커트 초과, 동점자 여부에 달림 |
| **도전권 이하** | — | 합격 가능성 낮음 |

**배수 위치** = 내 순위 ÷ 모집인원
- 배수 ≤ 1.0 → 확실권
- 배수 ≤ 1.2 → 유력권
- 배수 ≤ 합격배수 → 가능권
- 배수 > 합격배수 → 도전권

> **코드**: `src/lib/prediction.ts` → `calculatePrediction()`

## 7. 가산점 체계

### 7-1. 필기 단계 가산점 (취업지원/의사상자)

| 종류 | 비율 | 제한 |
|------|------|------|
| 취업지원대상자 | 과목별 만점의 **5%** 또는 **10%** | 선발인원 30% 초과 불가 |
| 의사상자 등 | 과목별 만점의 **3%** 또는 **5%** | 선발인원 10% 초과 불가 |

- 두 종류 중 **유리한 1개만** 선택 (중복 불가)
- 과락 과목에는 가산점 미적용
- **공채**: 100점 × 비율 × 3과목에 각각 적용 → 최대 30점 (10%)
- **경채**: 100점 × 비율 × 2과목에 각각 적용 → 최대 20점 (10%)

**코드**: `src/lib/scoring.ts` → `BonusType` enum, `BONUS_RATE_BY_TYPE`

### 7-2. 자격증 가산점 (최종 환산 단계 — 현재 수집만)

| 가산점 | 해당 자격 |
|-------|---------|
| 0점 | 해당 없음 |
| 2점 | 소방 관련 자격 하위 등급 등 |
| 4점 | 소방 관련 자격 상위 등급 등 |
| 5점 | 최상위 자격 등 |

> **중요**: 자격증 가산점은 **필기 finalScore에 포함되지 않습니다**.
> DB의 `Submission.certificateBonus`에 저장하지만 순위 계산에 미사용.
> 향후 최종환산 기능 구현 시 활용.

## 8. 성별 분리 선발 — 모집단 분리 규칙

순위 계산 시 어떤 응시자끼리 비교할지 결정합니다.

| 유형 | 모집단 분리 방법 |
|------|--------------|
| **공채** | 지역 + 유형 + **성별** 로 분리 |
| **구조** | 지역 + 유형 (성별 무관 — 남자만이라 의미 없음) |
| **소방학과** | 지역마다 다름 → quota의 `recruitAcademicCombined` 확인 |
| **구급** | 지역 + 유형 + **성별** 로 분리 |

### 소방학과 모집단 결정 로직

```
if (quota.recruitAcademicCombined > 0) {
  // 해당 지역은 양성(통합) 채용
  // → 남녀 모두 같은 모집단에서 경쟁
} else {
  // 해당 지역은 남녀 분리 채용
  // → 성별 기준으로 별도 모집단
}
```

## 9. 서비스 데이터 흐름

```
[관리자] 정답키 업로드 (/admin/answers)
    ↓
[사용자] OMR 답안 제출 (/exam/input)
    ↓
[시스템] 자동 채점
  - 과목별 원점수 = 정답수 × 배점
  - 과락 판정 (과목별 40% + 총점 60%)
  - 가산점 적용 → finalScore 저장
    ↓
[탭2] 내 성적 분석 (/exam/result)
  - 과목별 점수 차트
  - 원점수 / 백분위 / 석차 / 과락 여부
    ↓
[탭3] 합격예측 (/exam/prediction)
  - 동일 시험+지역+유형(+성별) 참여자 중 순위 계산
  - 배수 위치 = 내 순위 / 모집인원
  - 합격배수 대비 등급 판정 (확실/유력/가능/도전)
  - 경쟁자 목록 (이름 마스킹)
```
